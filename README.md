# Structura SwiftUI

`StructuraSwiftUI` is a Swift Package that collects reusable SwiftUI design-system building blocks for iOS apps.

## Requirements

- iOS 17+
- Swift 6.2
- Xcode with Swift Package Manager support

## Included Today

- Typography helpers via `STTypography`
- Asset-backed color palette via `STAsset.ColorPalette`
- Button styling
- Text field styling
- Toggle and checkbox styles
- Modal presentation helpers
- Toast and notification components

## Add To An App

Add this repository as a local Swift package in Xcode, then link the `StructuraSwiftUI` library product to your app target.

## Package Manifest

- Package name: `StructuraDesignSystemSwiftui`
- Library product: `StructuraSwiftUI`
- Target: `StructuraSwiftUI`
